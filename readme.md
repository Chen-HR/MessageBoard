# MessageBoard

## Description

Entrusted to use Python-Django to make a message board program.

All users (whether they have a account or not) can enter to create articles and leave comments on articles.

## Supported features

- [x] Account
  - [x] Signup
  - [ ] Edit
  - [x] Login
  - [x] Logout
- [x] Article
  - [x] Create
  - [ ] Edit
  - [x] List (HomePage)
  - [x] Content
    - [x] Message
      - [x] Create
      - [ ] Edit
      - [x] List
